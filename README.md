# Nepal Education Fund - Charity Website

A modern, responsive website for Nepal Education Fund, dedicated to providing financial support to underprivileged children's education in Nepal.

## 🌟 Features

### Core Sections
- **Home/Hero Section**: Eye-catching landing section with call-to-action
- **About Us**: Mission, vision, and impact statistics
- **Programs**: Six comprehensive programs:
  - Scholarship Program
  - Boarding Support
  - Learning Resources
  - Health & Nutrition
  - Digital Skills
  - Community Support
- **Impact Stories**: Real stories from supported children
- **Donation System**: Multiple donation options and custom amounts
- **Contact Section**: Get in touch with the organization
- **Responsive Design**: Works seamlessly on all devices

### Technical Features
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Smooth scrolling navigation
- ✅ Interactive donation modal
- ✅ Form validation
- ✅ Intersection Observer animations
- ✅ Counter animations for statistics
- ✅ Mobile-friendly hamburger menu
- ✅ Accessibility features
- ✅ Modern CSS with animations
- ✅ Cross-browser compatible

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No dependencies required - pure HTML, CSS, and JavaScript

### Installation

1. Clone the repository:
```bash
git clone https://github.com/edwardn340/Nepal-Charity.git
cd Nepal-Charity
```

2. Open the website:
```bash
# Simply open index.html in your browser
open index.html
```

Or deploy to a web server of your choice.

## 🔧 Customization

### To customize the organization name:
1. Replace "Nepal Education Fund" with your organization name
2. Update the footer information
3. Modify contact details

### To change colors:
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #dc143c;
    --secondary-color: #1e3a8a;
    --accent-color: #f59e0b;
    /* ... */
}
```

### To add more content:
1. Add new sections in `index.html`
2. Add corresponding styles in `styles.css`
3. Add interactivity in `script.js` if needed
